# Simple mergeSort implementation in C

## Build from source
### POSIX
Builds and runs tests and example
```
./build.sh test
./build.sh example 
```

### Windows MSVC
Builds and runs tests and example
```
.\build.bat && .\test 
.\build.bat && .\example
```

## Run binary
### POSIX
Run example
```
./example
```

### Windows
Run example
```
.\example.exe
```
