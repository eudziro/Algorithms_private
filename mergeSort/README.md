# Simple mergeSort implementation;

## Build from source
### POSIX
```
./build.sh example // runs example
...
./build.sh test //runs tests
```

### Windows MSVC
```
.\build.bat && .\example // runs example
...
.\build.bat && .\test // runs tests
```


## Run binary
### POSIX
```
./example // run example program
...
./test // run tests
```

### Windows
```
.\example.exe // run example program
...
.\test.exe // run tests
```
